# Aplicativo RSM Primer Orden

## 1. Nombre del proyecto

**RSM Primer Orden**

Aplicativo interactivo para el diseño, análisis y documentación de experimentos mediante metodología de superficie de respuesta de primer orden.

---

## 2. Objetivo general

Desarrollar una aplicación web interactiva que permita construir diseños experimentales de primer orden, analizar datos experimentales, ajustar modelos lineales, evaluar ANOVA, diagnosticar residuos, identificar factores relevantes mediante Pareto de efectos, proponer una ruta de ascenso más pronunciado y generar reportes ejecutivos.
